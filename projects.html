Meet App

This is a serverless, progressive web application (PWA) built with React using a test-driven development (TDD) technique. The application uses the Google Calendar API to fetch upcoming events. The user can search events by city and select the number of events displayed. Event stats are visualized in a pie chart and a scatter chart.
